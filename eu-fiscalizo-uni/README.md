# Projeto eu-fiscalizo-uni

Este é um projeto de desenvolvimento web moderno construído com foco em desempenho e uma experiência de usuário rica.

## Tecnologias Utilizadas

Este projeto foi construído utilizando um *stack* moderno de desenvolvimento frontend:

* **Vite:** Ferramenta de build extremamente rápida.
* **React:** Biblioteca para construção da interface de usuário (UI).
* **TypeScript:** Adiciona tipagem estática ao JavaScript, melhorando a escalabilidade e reduzindo erros.
* **Tailwind CSS:** Framework CSS *utility-first* para estilização rápida e responsiva.
* **shadcn-ui:** Coleção de componentes de UI acessíveis e reutilizáveis, baseados em React e Tailwind.

## Como Iniciar o Desenvolvimento Local

Para rodar este projeto no seu ambiente local (VS Code ou IDE de preferência), siga os passos abaixo.

### Pré-requisitos

Certifique-se de ter o **Node.js** e o **npm** instalados na sua máquina.

### Passos

1.  **Clone o Repositório:**

    ```bash
    git clone <URL_DO_SEU_REPOSITORIO>
    ```

2.  **Acesse o Diretório do Projeto:**

    ```bash
    cd eu-fiscalizo-uni
    ```

    *(Observação: Se você tiver uma subpasta de frontend ou client, navegue até ela antes do próximo passo.)*

3.  **Instale as Dependências:**
    Este comando baixa todos os pacotes e bibliotecas necessários definidos no `package.json`.

    ```bash
    npm install
    # ou npm i
    ```

4.  **Inicie o Servidor de Desenvolvimento:**
    Este comando inicia o servidor local com *hot-reloading* (recarregamento automático), permitindo que você veja as alterações no navegador em tempo real.

    ```bash
    npm run dev
    ```
